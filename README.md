# Sistema de Restaurantes

Projeto desenvolvido em Python para gerenciamento de restaurantes via terminal.

## Funcionalidades
- Cadastro de restaurantes
- Listagem de restaurantes
- Alteração de status
- Busca de restaurantes

## Tecnologias utilizadas
- Python 3

## Objetivo
Projeto criado para praticar:
- Funções
- Estruturas condicionais
- Loops
- Listas e dicionários
- Tratamento de erros
- Organização de código

## Como executar

```bash
python sistemaderestaurantes.py
```
